Goanna Log4E provides a port of the Jakarta Log4j Java logging framework. Goanna Log4E provides very flexible logging capabilities for Eiffel applications.

Goanna Log4E originates from the Goanna Web Application Framework.

Click the Documentation link above for installation, compilation and usage instructions.

Full class documentation is also available.

15 July 2002 - Release 1.0.

The Goanna Log4E project is hosted by SourceForge which provides a public CVS repository, mailing lists, file store and much more. Links to the latest release packages can be found on the Downloads page.

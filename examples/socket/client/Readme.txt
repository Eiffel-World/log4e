Log4E Socket Client Example
---------------------------

This example writes a series of logging events to a socket as 
specified on the command line.

Once compiled, start the client by specifying the host and port to write
events to. eg:

	log4e_socket_client localhost 9000

Additional Requirements
-----------------------

ISE Eiffel 5.1 or greater.
ISE EiffelNet library.
